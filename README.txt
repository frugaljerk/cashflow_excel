Quickstart Guide
----------------

1. Download CSV files for each TD Webbroker account to stocks. (GUI to be implemented in the future)
2. Change the currency exchange rate on the CashFlow_2021.xlsx file manually (To implement auto update with API)
3. Run main.py to update numbers from the CSV files to the master CashFlow_2021.xlsx. New sheet with the name of current month will be generated.
4. Manually fill in the income and expense section in the master CashFlow_2021.xls. (To implement autofill in the future)
